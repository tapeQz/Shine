# What does it do?
Adds **selective Bloom** to the game, **no Iris required**. 

Little to **no performance loss** when enabled, with support for [Sodium](https://modrinth.com/mod/sodium).

**Highly customizable** config screen with [Mod Menu](https://modrinth.com/mod/modmenu), so change it to your preference. 

Requires [YACL.](https://modrinth.com/mod/yacl)

# Recommendation
I highly recommend you use a resource pack alongside this that adds **Emissive Elements**, such as [**Emissive Shaders.**](https://modrinth.com/resourcepack/emissive+shaders)
